# ANTIGRAVITY.md - 碧蓝幻想Chrome拡張「太郎」日本語化プロジェクト憲法

## 1. プロジェクト目的
本プロジェクトの目的は、中国語ベースで開発されている『グランブルーファンタジー』用Chrome拡張機能「太郎（Tarou）」を、日本のプレイヤーが違和感なく利用できるよう、**直接的なハードコード書き換え**によって日本語化することである。

## 2. ターゲット
- 日本の『グランブルーファンタジー』プレイヤー
- 日本語のゲームUIに慣れ親しんでいるユーザー

## 3. 技術スタック
- **Core**: Vue 3 + TypeScript + Vite
- **UI**: Element Plus + UnoCSS
- **Communication**: webext-bridge
- **Strategy**: ソースコード内の文字列（Vueテンプレート、TypeScript、定数等）の直接置換

## 4. MVP（最小構成）の範囲
まずはユーザーが最も頻繁に利用する以下の画面の日本語化を優先する。
1. **サイドパネル (Side Panel)**: 編成（Party）、戦闘（Battle）、ドロップ（Drop）等
2. **ポップアップ (Popup)**: 基本設定、サポート召喚石選択等
3. **戦闘パネル (Combat Panel)**: ターン数、NPCカウント等

## 6. 運用ルール
- 翻訳はゲーム内の用語（例：武器スキル、加護、召喚石など）に厳密に合わせる。
- ソースコードのロジックを壊さないよう、文字列部分のみを慎重に置換する。
- 書き換え前に、対象ファイルと内容をユーザーに提示し「y/n」の確認を取る。
- 各ファイルまたは各機能の変更ごとに、Gitでコミットを実行する。
- **ビルド・配布**: 今後は配布用の ZIP 作成は不要とし、`dist` フォルダのビルド更新を優先して行う。
